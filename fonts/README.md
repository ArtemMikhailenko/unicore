# Fonts

## Required Fonts

### HYWenHei-85W
- **File needed**: `HYWenHei-85W.ttf`
- **Usage**: Heading font for "Unicore" title
- **Weight**: 700 (Bold)

Place the font file in this directory: `/fonts/HYWenHei-85W.ttf`

If you don't have this font, you can:
1. Use a similar bold display font
2. Or temporarily use the fallback (system-ui, sans-serif)

### Source Sans 3
- **Loaded from**: Google Fonts (automatic)
- **Usage**: Body text and descriptions
- **Weights**: 400 (Regular), 600 (SemiBold)
